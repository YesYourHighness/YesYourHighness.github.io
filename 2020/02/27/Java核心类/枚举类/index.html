创建、使用

为了解决上述的两个问题，Java创建了枚举类

  • 编译器自动检测某个值是不是在枚举的集合内
  • 不同用途的枚举需要不同的类型来标记

创建一个枚举类，使用enum

使用

优点

使用枚举类比最开始提到的static final主要有以下的优点：

  1. enum的常量有类型，类型是类名，而定义的static final是int类的，编译器会自动检查类型比较错误

  2. 因为他们类型不一样,不同类型的枚举不能互相赋值或者比较

比较

枚举类是引用类型，引用类型我们都说得使用equals()方法比较

但是枚举类可以使用==比较，因为在java中每个枚举常量在JVM中只有一个唯一示例

方法

枚举类的特点

  • 枚举类继承自java.lang.Enum，并且无法继承
  • 无法使用new创建实例
  • 定义的每个实例都是引用类型的唯一实例
  • 可以在switch语句中使用
