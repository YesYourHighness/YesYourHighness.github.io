Properties

Properties

继承与Hashtable，来表示一个持久的属性集，它用键值结构存储数据，每个键及其对应值都是一个字符串，

该类也被许多Java类使用，比如获取系统属性时，System.getProperties方法就是返回一个Properties对象

  • Properties集合是唯一一个与IO流相结合的集合

    可以使用Properties集合中的方法store，把集合中的临时数据，持久化写入到硬盘中存储

可以使用其中的`load`方法，把硬盘存储的文件（键值对），读取到集合中使用
