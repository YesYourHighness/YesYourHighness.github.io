Properties

Properties 属性映射：是一种存储键值对的数据结构，在java中用来存储配置信息

特性：

  1. 继承自Hashtable，每个键及其对应值都是一个字符串
  2. 映射可以很容易的存入文件以及从文件中加载
  3. 有一个二级表保存默认值
  4. 该类也被许多Java类使用，比如获取系统属性时，System.getProperties方法就是返回一个Properties对象

Properties集合是唯一一个与IO流相结合的集合

注意：

  1. 存储键值对的文件中，键与值默认的连接符号可以使用 -，空格（其他符号）
  2. 存储键值对的文件中，可以使用#进行注释，被注释的键值对不会再被读取
  3. 存储键值对的文件中，键与值默认都是字符串，不用再加引号

缺点

  • 某些操作系统没有主目录的概念，所以很难找到一个统一的配置文件位置

可以用System.out.println(System.getProperties().getProperty("user.home"));来获取用户主目录

  • 关于配置文件的命名没有标准约定，用户安装多个java应用时，很容易发生命名冲突
