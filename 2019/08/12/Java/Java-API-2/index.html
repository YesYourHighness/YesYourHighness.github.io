字符串常量池

在java虚拟机堆中，字符串有一个专门的常量池

直接写上双引号的字符串，就在字符串常量池中

new创建的字符串不在常量池内

对于引用类型来说，等于号是对地址进行的比较

特点：

  1. 字符串的内容不可以改变
  2. 字符串内部是可以共享使用的，相同加了双引号的字符串，如果内容都相同，他的地址也是来自一块地址的
  3. 字符串的底层原理是byte字节数组
