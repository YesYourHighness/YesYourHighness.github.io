    同步技术的原理：

使用了一个锁对象，这个锁对象叫同步锁，也叫对象锁，也叫对象监视器

当发生多线程分享共有资源时，抢到cpu运行权的时候，

运行到synchronize时，检查是否有锁对象，
有，
就会获取到锁对象并进入到同步过程中，运行完归还锁对象。

没有，就会等待直到锁对象被归还。
